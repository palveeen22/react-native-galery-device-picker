# react-native-upload-local

# How to Use
- `npx expo install expo-image-picker`
- expo/app.json`{
  "expo": {
    "plugins": [
      [
        "expo-image-picker",
        {
          "photosPermission": "The app accesses your photos to let you share them with your friends."
        }
      ]
    ]
  }
} `
- import `import * as ImagePicker from 'expo-image-picker'`
